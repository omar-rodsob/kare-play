<h3 align="center">Kare Play</h3>

<h3 align="center">Steps</h3>
<p>Clone the repo</p>

  ```
  git clone git@github.com:omar-rodsob/kare-play.git
 ```
<p>Acces to the repo</p>

 ```
cd kare-play/
 ```

<p>Run Install </p>

```
npm install
```

<p>Run the test</p>

 ```
npm test
 ```
